훈민정음 해례본[갑]은 영어와 특수기호 없이 순한글로 이루어진 프로그래밍 언어입니다.  
모든 문법이 한글로 이루어져 있으며, 훈민정음·조선왕조실록의 고어체를 바탕으로 설계되었습니다.

---

## 이름의 뜻

**훈민정음 해례본**은 세종대왕이 1443년에 만든 훈민정음의 창제 원리와 사용법 등을 해설해 놓은 책의 제목입니다.
어려운 것을 쉽게 풀어쓴다는 뜻처럼, 이 언어는 어려운 프로그래밍 언어를 한글로 풀어쓰고자 하였습니다.  
**[갑]** 은 천간(天干)의 첫 번째로, 이 언어의 첫 번째 판임을 뜻합니다.

## 문법

### 변수 선언

`정수`, `실수`, `말` 은 각각 파이선(派以善)의 `int`, `float`, `str` 을 대체합니다. 변수명 뒤에 조사 `은` / `는` 을 띄어 써서 값을 대입합니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `정수 나이 는 이십` | `age = 20` | 정수형 변수 선언 |
| `실수 키 는 백칠십 점 오` | `height = 170.5` | 실수형 변수 선언 |
| `말 이름 은 이르되 홍길동 하니라` | `name = "홍길동"` | 문자열 변수 선언 |

<br>

- `은` / `는` 둘 다 허용
- 실수는 `점` 을 기준으로 앞은 정수 덩어리, 뒤는 한 글자씩 소수 자릿수로 읽는다

<br>

> **왜 따옴표가 없나요?**  
> 일반적인 프로그래밍 언어에서 문자열을 감쌀 때 쓰는 `"` 는 서양에서 들어온 기호입니다.  
> 훈민정음 해례본[갑]은 고증을 위해 기호를 사용하지 않으며, 대신 `이르되 ~ 하니라` 로 문자열의 시작과 끝을 나타냅니다.

아라비아 숫자 대신 한자식 수사를 사용합니다.

<br>

| 숫자 | 표기 |
|------|------|
| 0 | `영` |
| 1 | `일` |
| 10 | `십` |
| 100 | `백` |
| 1,000 | `천` |
| 10,000 | `만` |
| 100,000 | `십만` |
| 100,000,000 | `일억` |
| 1,000,000,000,000 | `일조` |

### 출력

`이르라` 는 파이선(派以善)의 `print()` 를 대체합니다. 변수명을 바로 뒤에 적으면 변수 값을, `이르되 ~ 하니라` 형식으로 적으면 문자열을 직접 출력합니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `이르라 이름` | `print(name)` | 변수 값 출력 |
| `이르라 이르되 안녕 하니라` | `print("안녕")` | 문자열 직접 출력 |

<br>

#### 문자열 안에 변수 넣기

`부르되 ~ 하고` 는 파이선(派以善)의 f-string(`f"{변수}"`)을 대체합니다. `이르되 ~ 하니라` 문자열 안에서 변수를 삽입하고 싶은 자리에 씁니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `이르되 안녕 부르되 이름 하고 님 하니라` | `f"안녕 {name} 님"` | 문자열 안에 변수 삽입 |

```
이르라 이르되 안녕하세요 부르되 이름 하고 님 하니라
```

출력: `안녕하세요 홍길동 님`

<br>

#### 개행

`개행` 은 파이선(派以善)의 `\n` 을 대체합니다. `이르되 ~ 하니라` 문자열 안에서 줄을 바꾸고 싶은 자리에 씁니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `개행` | `\n` | 줄바꿈 |

```
이르라 이르되 첫째줄 개행 둘째줄 하니라
```

### 사칙연산

`합하다`, `감하다`, `곱하다`, `나누다`, `남다` 는 각각 파이선(派以善)의 `+`, `-`, `*`, `//`, `%` 를 대체합니다. `변수 연산자 변수` 형식으로 씁니다. `나누다` 는 몫만 구하는 정수 나누기입니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `갑 합하다 을` | `a + b` | 더하기 |
| `갑 감하다 을` | `a - b` | 빼기 |
| `갑 곱하다 을` | `a * b` | 곱하기 |
| `갑 나누다 을` | `a // b` | 나누기 (몫) |
| `갑 남다 을` | `a % b` | 나머지 |

```
정수 결과 는 갑 합하다 을
```

---

### 조건문

`만약 ~ 하되`, `아니하고 만약 ~ 하되`, `아니하면 하되` 는 각각 파이선(派以善)의 `if`, `elif`, `else` 를 대체합니다. 파이선(派以善)과 달리 들여쓰기만으로는 블록이 끝나지 않으며 반드시 `끝이라` 로 닫아야 합니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `만약 ~ 하되` | `if ~:` | 조건이 참이면 실행 |
| `아니하고 만약 ~ 하되` | `elif ~:` | 앞 조건이 거짓이고 이 조건이 참이면 실행 |
| `아니하면 하되` | `else:` | 앞 모든 조건이 거짓이면 실행 |
| `끝이라` | (들여쓰기로 대체) | 블록 끝 |

```
만약 나이 가 이십 보다 클 하되
    이르라 이르되 어른이옵니다 하니라
아니하고 만약 나이 가 십 보다 클 하되
    이르라 이르되 청소년이옵니다 하니라
아니하면 하되
    이르라 이르되 어린이옵니다 하니라
끝이라
```

<br>

#### 비교 연산자

`변수 가 값 보다 ~` 또는 `변수 가 값 과 ~` 형식으로 씁니다. 파이선(派以善)의 `<`, `>`, `<=`, `>=`, `==`, `!=` 를 대체합니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `나이 가 이십 보다 작을` | `age < 20` | 미만 |
| `나이 가 이십 보다 클` | `age > 20` | 초과 |
| `나이 가 이십 보다 작거나 같을` | `age <= 20` | 이하 |
| `나이 가 이십 보다 크거나 같을` | `age >= 20` | 이상 |
| `나이 가 이십 과 같을` | `age == 20` | 같음 |
| `나이 가 이십 과 같지않을` | `age != 20` | 다름 |

### 횟수 반복

`되풀이 ~ 부터 ~ 전까지 하되` 는 파이선(派以善)의 `for i in range():` 를 대체합니다. `전까지` 는 해당 값을 포함하지 않습니다. 반복 변수를 지정하면 현재 반복 횟수를 해당 변수에서 쓸 수 있습니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `되풀이 영 부터 오 전까지 하되` | `for i in range(0, 5):` | 횟수 반복 |
| `되풀이 일 부터 십 전까지 갑 하되` | `for 갑 in range(1, 10):` | 반복 변수 사용 |

```
되풀이 일 부터 십 전까지 갑 하되
    이르라 갑
끝이라
```

---

### 조건 반복

`동안 ~ 하되` 는 파이선(派以善)의 `while:` 을 대체합니다. 조건이 참인 동안 블록을 계속 실행하며, 무한 루프 방지를 위해 최대 만 번까지만 실행됩니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `동안 나이 가 오 보다 작을 하되` | `while age < 5:` | 조건이 참인 동안 반복 |

```
동안 나이 가 오 보다 작을 하되
    이르라 나이
    정수 나이 는 나이 합하다 일
끝이라
```

### 함수

`함수 ~ 하되` 는 파이선(派以善)의 `def:` 를 대체합니다. 매개변수는 `받되` 뒤에 나열하며, 반환값은 `바치다` 로 돌려줍니다. 함수를 호출할 때는 `행하라` 를 씁니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `함수 이름 받되 갑 을 하되` | `def 이름(갑, 을):` | 함수 선언 |
| `받되 갑 을` | `갑, 을` | 매개변수 |
| `바치다 결과` | `return 결과` | 반환값 |
| `행하라 이름 받되 갑 을 하되` | `이름(갑, 을)` | 함수 호출 |
| `정수 답 은 행하라 이름 받되 갑 을 하되` | `답 = 이름(갑, 을)` | 반환값 받기 |

```
함수 더하다 받되 갑 을 하되
    정수 결과 는 갑 합하다 을
    바치다 결과
끝이라

정수 답 은 행하라 더하다 받되 삼 오 하되
이르라 답
```

### 주석

`사신은 논한다` 는 파이선(派以善)의 `#` 을 대체하는 한 줄 주석입니다. 여러 줄 주석은 `사신은 논하되` 로 시작해서 `논하기를 마치다` 로 끝냅니다. 조선왕조실록에서 사관(史臣)이 의견을 달 때 실제로 쓰던 표현에서 따왔습니다.

<br>

| 훈민정음 | 파이선(派以善) | 역할 |
|---------|-------------|------|
| `사신은 논한다 내용` | `# 내용` | 한 줄 주석 |
| `사신은 논하되 ~ 논하기를 마치다` | `""" ~ """` | 여러 줄 주석 |

```
사신은 논한다 이것은 한 줄 주석입니다

사신은 논하되
    이것은 여러 줄에 걸친
    긴 주석이옵니다
논하기를 마치다
```

### 오류 메시지

실행 중 오류가 발생하면 `아 슬프다` 로 시작하는 메시지가 출력됩니다. 조선왕조실록의 탄식 표현에서 따왔습니다.

<br>

| 상황 | 메시지 |
|------|--------|
| 없는 변수 참조 | `아 슬프다 변수 나이 가 존재하지 아니하옵니다` |
| 잘못된 문법 | `아 슬프다 알 수 없는 문법이옵니다 하오니 ~ 을 살펴보시옵소서` |
| 영으로 나누기 | `아 슬프다 영으로 나눌 수 없사옵니다` |
| 무한 반복 | `아 슬프다 무한 되풀이가 감지되었사옵니다` |
| 없는 함수 호출 | `아 슬프다 함수 이름 이 존재하지 아니하옵니다` |
